# ytdl

A script to handle a quick-and-easy usage for yt-dlp.


# How to install

## MacOS (brew)
Run the following commands in your terminal:

```bash
brew tap zeroonelabs/tap
brew install ytdl
```

## Linux

Download the script, save it to `~/bin/ytdl`, and run `chmod +x ~/bin/ytdl` to make the script executable.
